    Features
    • 📝 Quick notes while browsing — always one click away
    • 🖱️ Right-click any text → Add to Memo
    • 💾 Save with one click & edit anytime
    • ✏️ Custom titles & 📌 pin favorites to top
    • 🔍 Search & 🔀 sort your saved memos
    • 🔤 Font size control: Large / Larger / Huge
    • 🎨 Font colors: Black, Red, Blue
    • 🖼️ Paste images straight into a note
    • 🌗 Themes: Light / Dark / Warm
    • 📥 Export memos as HTML or 📄 Markdown
    • 💾 Back up & restore all memos (JSON)
    • ⌨️ Keyboard shortcuts: Save & Search
    • ⚙️ Settings tab to tailor how it works
    New in 1.2.0
    • 💾 Full backup & restore — export/import all memos as JSON
    • ⌨️ Keyboard shortcuts — Save (⌘/Ctrl+S) & Search (⌘/Ctrl+F), toggleable in Settings
    • 🖥️ OS-native system fonts — SF on Mac, Segoe UI on Windows
