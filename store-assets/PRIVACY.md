# Simple Side Note — Privacy Policy

**Extension Name:** Simple Side Note
**Developer:** Swyoon Labs
_Last updated: 2026-06-03_

## 1. Overview

Simple Side Note is a Chrome extension that allows users to write and manage simple notes
in the Chrome Side Panel.

We respect user privacy. This extension is designed to store user-created notes locally
and does **not** collect, transmit, sell, or share personal information.

## 2. Information We Collect

Simple Side Note does not collect personal information from users.

The extension may store the following data **locally in the user's browser**:

- Notes written by the user
- Extension settings and preferences

The extension does **not** collect:

- Names
- Email addresses
- Passwords
- Payment information
- Location information
- Browsing history
- Website content
- Analytics data
- Tracking identifiers

## 3. How User Data Is Used

Locally stored notes and settings are used only to provide the extension's note-taking
functionality. The data is not used for advertising, analytics, profiling, or any
unrelated purpose.

## 4. Data Storage

User notes and preferences are stored using Chrome's Storage API. This data remains in the
user's browser environment and is not sent to any external server operated by Swyoon Labs
or any third party.

## 5. Data Sharing

Simple Side Note does not sell, rent, transfer, disclose, or share user data with third parties.

## 6. Remote Code and External Services

Simple Side Note does not use remote code execution. The extension does not send user-created
notes to external servers or third-party services.

## 7. Chrome Permissions

- **storage** — save notes and user preferences locally in Chrome.
- **sidePanel** — display the note-taking interface in Chrome's Side Panel.
- **contextMenus** — provide quick access to note-related actions from Chrome's context menu.

## 8. User Control

Users can edit or delete their notes within the extension. Users may also remove all locally
stored extension data by uninstalling the extension or clearing the extension's storage data
in Chrome.

## 9. Children's Privacy

Simple Side Note is not directed to children under the age of 13. The extension does not
knowingly collect personal information from children.

## 10. Changes to This Privacy Policy

This Privacy Policy may be updated when the extension's features or data handling practices
change. Any updates will be posted with a revised "Last Updated" date.
