# Simple Side Note v1.2.0 — 이제 메모를 잃어버릴 걱정 없이

> 브라우저 사이드바에 딱 붙어 있는 가벼운 메모장, **Simple Side Note**의 1.2.0 업데이트를 소개합니다. 이번 버전의 핵심은 한 문장으로 **"데이터를 지키고, 손은 덜 쓰게"** 입니다.

---

## 📝 한눈에 보기

| | 내용 |
|---|---|
| 💾 **백업 & 복원** | 모든 메모를 JSON 파일 하나로 내보내고, 언제든 다시 불러오기 |
| ⌨️ **키보드 단축키** | `⌘/Ctrl+S` 저장, `⌘/Ctrl+F` 검색 — 설정에서 켜고 끄기 가능 |
| 🖥️ **OS 시스템 폰트** | Mac은 SF, Windows는 Segoe UI로 자동 적용 |

## 💾 1. 백업 & 복원 — 이번 업데이트의 주인공

메모가 쌓일수록 커지는 걱정이 하나 있죠. **"이거 날아가면 어떡하지?"**

기존에는 메모를 하나씩 HTML·Markdown으로 내보낼 수는 있어도, **전체를 한 번에 백업할 방법이 없었습니다.** 확장을 실수로 삭제하거나 PC를 바꾸면 그동안 쌓은 메모가 사라질 수 있었어요.

1.2.0부터는 **⚙️ Settings 탭 → Backup & Restore** 에서 두 버튼으로 해결됩니다.

- **📤 Export all** — 저장된 모든 메모(제목·내용·핀·날짜)를 날짜가 붙은 JSON 파일 하나로 내려받습니다. 예: `simple-side-note-backup-2026-07-10.json`
- **📥 Import** — 백업 파일을 불러와 지금 메모에 **합칩니다.**

### 안심하고 써도 되는 이유

가져오기(Import)는 **절대 기존 메모를 지우지 않습니다.** 오직 추가만 해요. 게다가:

- **같은 메모는 중복으로 안 들어갑니다.** 같은 백업 파일을 두 번 불러와도 깔끔합니다.
- 내부적으로 메모 식별자가 겹치면 자동으로 새로 부여해 **데이터가 꼬이지 않습니다.**
- 저장 공간이 부족한 경우처럼 저장이 실패하면 **조용히 넘어가지 않고 알려줍니다.**

> 💡 **팁:** 중요한 메모가 쌓였다면, 가끔 **Export all** 한 번으로 백업 파일을 만들어 두세요. 그 파일만 있으면 어떤 상황에서도 **Import**로 되살릴 수 있습니다.

## ⌨️ 2. 키보드 단축키 — 마우스 없이 저장하고 검색하기

이제 손을 키보드에서 떼지 않고 메모를 다룰 수 있습니다.

| 동작 | macOS | Windows / Linux |
|---|---|---|
| 저장 | `⌘ S` | `Ctrl + S` |
| 검색 | `⌘ F` | `Ctrl + F` |

- 운영체제에 맞춰 **수식키가 자동으로 바뀝니다** — Mac에서는 `⌘`, 그 외에는 `Ctrl`. 버튼 위에 마우스를 올리면 툴팁으로 해당 단축키를 확인할 수 있어요.
- `⌘/Ctrl+F` 는 브라우저 기본 찾기 대신 **저장된 메모 검색창으로 바로 이동**합니다.
- 브라우저 기본 동작을 그대로 쓰고 싶다면? **⚙️ Settings 탭에서 단축키를 끌 수 있습니다.** 필요할 때만 켜세요.

## 🖥️ 3. OS 시스템 폰트 — 어느 OS에서든 자연스럽게

메모장이 브라우저·운영체제와 겉도는 느낌이 나지 않도록, UI와 편집 영역이 **각 OS의 기본 시스템 폰트**를 사용하도록 바꿨습니다.

- **Mac** → San Francisco (`-apple-system`)
- **Windows** → Segoe UI
- **Linux** → Roboto 등

별도 설정 없이 열기만 하면, 지금 쓰는 OS에 가장 잘 어울리는 글꼴로 표시됩니다. 내보낸 HTML 파일과도 이제 글꼴 인상이 일치해요.

- 폰트 색상 버튼(검정/빨강/파랑)의 툴팁이 한국어로만 표시되던 부분을 **영어(Black / Red / Blue)로 통일**해, UI 전체의 언어 일관성을 맞췄습니다.
